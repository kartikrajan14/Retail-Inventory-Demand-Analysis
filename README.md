# Retail Inventory Demand Analysis

## Project Overview
This project analyzes retail inventory and demand patterns to identify stockout risks, sales performance, and the impact of discounts. The analysis is performed using Python, SQL, and Power BI.

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib)
- SQL (SSMS)
- Power BI
- GitHub

## Project Structure
data/ → cleaned dataset
python/ → Jupyter notebooks for analysis
sql/ → SQL queries used for analysis
powerbi/ → Power BI dashboard file
screenshots/ → Dashboard and visual screenshots

## Key Analysis Performed
- Stockout risk analysis by product and category  
- Impact of discounts on sales  
- Regional sales performance  
- Inventory level and demand trend analysis  

## Power BI Dashboard
The Power BI dashboard provides insights into:
- Inventory levels
- Sales trends
- Stockout risk
- Regional performance

## How to Use
1. Review the cleaned dataset in the `data` folder.
2. Explore the Python notebook for analysis steps.
3. Open the Power BI (.pbix) file to interact with the dashboard.
4. Refer to SQL queries in the `sql` folder.
